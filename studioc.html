Studio C

Role: Graphic Designer

Studio C is an in-school web design firm run by the senior Web Design class each year. The firm is made up of several departments: content, graphics, coding, marketing, and testing. Studio C works with the community to create non-profit websites to better convey their message.

NACTE Logo Entry - Fall 2016

This was an assignment in my Web Development & Design I class. We had to create an informational website for Dante's Inferno by Dante Aligheri and represent his other works as a client project. This was done in a group.

I was in charge of doing the HTML and helping to come up with designs. I followed the design of the mockup and site map and collaborated with my team members. It was a great learning experience that helped improve my knowledge of code. I learned who I worked well with and what my strengths are when it comes to code. I used Sublime Text.
