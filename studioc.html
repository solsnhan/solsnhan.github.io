Studio C

Role: Graphic Designer

Studio C is an in-school web design firm run by the senior Web Design class each year. The firm is made up of several departments: content, graphics, coding, marketing, and testing. Studio C works with the community to create non-profit websites to better convey their message.

Here are some of the projects that I have been involved with.

x

NACTE Logo Entry - Fall 2016

This is my entry for the NACTE 2017 logo design contest. The goal was the incorporate the theme of "90 years of innovation: Look Back and Move Forward."

I used a lightbulb as the shape is most associated with the concept of ideas. I added the arrows to match the concept of "Look Back and Move Forward" and then added the name and date to the side. I created this with Adobe Illustrator.

x

Web Olympics Logo - Winter 2016

This is the logo design for the Web Olympics, an event held by the SWCTA web design class. The event was to showcase the knowledge of the students in the program from all grade levels in a variety of challenges and activities.

The rings are a play-off of the traditional Olympics' logo and the symbols are the ones associated with HTML5, CSS3, and JavaScript. I designed this with Adobe Illustrator.

x

Nevada PTA Mockup - Spring 2017

This is a mockup for the Nevada PTA app design. The final app received the Nevada PTA President's Special Award at the NVPTA conference in April.

I used the colors given to me by Nevada PTA to create an app that has a simple layout and clean design. This was created in Adobe Illustrator.

x

We Create Mockup - Spring 2017

We Create is a project from a teacher that was chosen by Google to come up with a project to help students. It is a resource that hosts images and allows users to correctly use and cite them. This is the mockup for the design of the site.

There was already a pre-existing site and I used that as inspiration. I chose the colors to be bright and bold to capture the attention of users (those of who are students). This was created in Adobe Illustrator.
